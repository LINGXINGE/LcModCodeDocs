AgentManager

命名空间: Global
AgentManager 类负责管理所有 Agent 模型的创建、激活和停用。
← 返回首页

公共属性

instance
属于/类型：AgentManager
获取 AgentManager 的单例实例
agentListSpare
属于/类型：List<AgentModel>
备用 Agent 模型列表
deletedAgent
属于/类型：List<long>
永久删除的 Agent 列表
customAgent
属于/类型：List<CustomizedAgentData>
自定义 Agent 数据列表
agentMaxCount
属于/类型：int
允许的最大 Agent 数量（常量：50）
