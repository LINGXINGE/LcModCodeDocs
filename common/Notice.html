Notice

命名空间: Global

通知系统类，实现观察者模式。用于游戏内的事件通知和消息传递，支持订阅、取消订阅、发送通知等功能。

← 返回首页

公共属性

instance
属于/类型：Notice
获取单例实例（静态）

重要方法

public void Observe(string notice, IObserver observer)
订阅通知，使用IObserver接口
public int Observe(string notice, NoticeReciever observer)
订阅通知，使用回调函数，返回通知ID
public void Remove(string notice, IObserver observer)
取消订阅通知
public void Remove(string notice, int noticeId)
通过通知ID取消订阅
public void Send(string notice, params object[] param)
发送通知给所有订阅者

使用示例

// 订阅通知
int noticeId = Notice.instance.Observe("OnCreatureEscape", (param) => {
    CreatureModel creature = param[0] as CreatureModel;
    Debug.Log("异想体逃脱: " + creature.name);
});

// 发送通知
Notice.instance.Send("OnCreatureEscape", creatureModel);

// 取消订阅
Notice.instance.Remove("OnCreatureEscape", noticeId);
